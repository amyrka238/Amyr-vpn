2. На своём компьютере:

   git init
   git add .
   git commit -m "Initial commit: SubManager v2.0"
   git branch -M main
   git remote add origin https://github.com/YOUR_USERNAME/submanager.git
   git push -u origin main

3. Готово! Репозиторий на GitHub
